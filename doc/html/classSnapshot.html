Detailed Description

Collects all data that describes a Snapshot and provides a method to insert it into the scene. (Not the smartest implementation, should probably be constructed given the scene manager and do the object creation by itself. Static counter would help to identify total number of Snapshots to link to individual materials, textures etc.)
