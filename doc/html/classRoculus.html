Detailed Description

class to take care of the scene and object creation. This class fills the scene with objects and in most cases creates the geometries manually. It has a method to parse room recordings in folders in order to include them into the scene. Though it is the nominal main class, the majority of the implementation is found in the BaseApplication class.
