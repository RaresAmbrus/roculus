Detailed Description

Ogre BaseApplication class The BaseApplication class is the central class in the project. It brings all systems together: *initialization and control of the rendering process *handling keyboard/mouse input *connection to ROS for joystick input, image sensor data, waypoint data, etc. *cleaning up after the application finished.
