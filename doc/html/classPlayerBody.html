Detailed Description

Handles the PlayerBody motion. Extension class to handle the player motion independent from the Oculus Rift head motion. The Oculus motion happens on top of this, which is achieved by making the PlayerBody's scene node the parent for the Oculus handler. Since this was one of my first implementations (very unclean) this step, in fact, happens in BaseApplication.
